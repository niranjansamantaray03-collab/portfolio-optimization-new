# 📈 Portfolio Optimization using Modern Portfolio Theory

## 📌 Overview
This project builds a portfolio optimization model using Python to maximize returns while minimizing risk.

## 🚀 Features
- Uses real stock market data
- Calculates expected return and risk
- Optimizes portfolio using Sharpe Ratio
- Visualizes Efficient Frontier

## 🧠 Concepts Used
- Modern Portfolio Theory (MPT)
- Risk-Return Tradeoff
- Covariance Matrix
- Optimization Techniques

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- SciPy
- yFinance
